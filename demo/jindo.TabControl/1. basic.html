JindoJS는 Javascript Application 제작시에 반복적으로 발생하는 로직을 컴포넌트화하여 결과물의 품질을 향상시키기 위한 목적으로 작성되었습니다. NAVER에서 서비스하고 있는 수많은 페이지안의 UI들이 JindoJS를 사용하여 개발되었습니다. 그런만큼 JindoJS는 크로스 브라우징에 있어서 거의 완벽한 호환성을 입증하고 있습니다.
JindoJS Component는 JindoJS 를 기반으로 개발된 재사용 가능한 독립적인 자바스크립트 모듈입니다. JindoJS는 Javascript Application 제작시에 반복적으로 발생하는 로직을 컴포넌트화 하여 결과물의 품질을 향상시키기 위한 목적으로 작성되었습니다. NAVER에서 서비스하고 있는 수많은 페이지안의 UI들이 JindoJS를 사용하여 개발되었습니다. 그런만큼 JindoJS는 크로스 브라우징에 있어서 거의 완벽한 호환성을 입증하고 있습니다.
Jindo Mobile Component(JMC)는 JindoJs 기반으로 개발된 재사용 가능한 독립적인 자바스크립트 모듈로써, 국내 모바일 환경에 최적화된 모바일 컴포넌트입니다. 오픈 소스인 JMC는 사내 및 사외 Javascript Mobile Application 제작시에 반복적으로 발생하는 로직을 컴포넌트화하여 결과물의 품질을 향상시키기 위한 목적으로 작성되었습니다. NAVER의 수많은 모바일 서비스에서 사용하고 있을 정도로 품질과 안정성이 뛰어난 JMC는 120여종의 샘플과 상세한 가이드를 제공하고 있어서, 초급자도 손쉽게 고품질의 UI를 개발할 수 있습니다.
