jindo.Pagination을 이용한 간단한 튜토리얼

이번 샘플에서는 간단한 jindo.Paginantion을 만들어본다.

1. jindo.Pagination 생성하기

jindo.Pagination을 만들 엘리먼트와 페이지 형태로 보여줄 항목 10개를 jindo.Pagination에 파라미터로 넘겨줍니다.

2. 옵션변경하기

한 페이지에 1개의 항목을 보여주고 한 페이지에 페이지 목록은 3개의 페이지 목록이 보이도록 설정합니다.

이전 다음
3. 이벤트 연동하기

페이지가 변경되는 경우마다 화면에 노출되는 상품 이미지가 변경되는 코드를 추가합니다. 상품이미지는 N_000.jpg, N_001.jpg에서 N_009.jpg까지 총 10개의 이미지를 가지고 있습니다.

jindo.Pagination의 move 이벤트에 페이지가 변경되었을 때의 동작을 추가합니다.

  1. jindo.Pagination().attach() 함수에 move 이벤트를 어떻게 처리할 지를 정의한 함수를 넘겨줍니다.
  2. 페이지 변경 시에 넘겨주는 페이지 번호를 받기 위해서 파라미터에 oCustomEvent 를 추가합니다.
  3. 넘겨주는 페이지 값이 1부터 시작하나 상품 이미지는 0부터 시작하므로 1을 빼주는 코드를 추가합니다.
  4. 이미지 태그의 src를 해당 페이지의 상품 이미지로 교체해줍니다.

최종 작성한 코드는 다음의 형태가 됩니다.
